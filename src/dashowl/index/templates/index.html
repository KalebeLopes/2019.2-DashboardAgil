Administre o andamento de seus projetos!

Nossa plataforma foi criada com o intuito de acompanhar em tempo real os repositórios do GitHub tirando métricas e indicadores das atividades feitas individualmente e pelo time em forma de gráficos.




É muito fácil e prático de usar…

Entre com seu login do GitHub, selecione o repositório que deseja analisar e pronto!




Dashboard Ágil

Nossas métricas para a criação do dashboard são todas baseadas no método Ágile. Esse método vem sendo muito utilizado no desenvolvimento de software. Além disso, um projeto do GitHub traz muitas informações difusas ou sem nenhuma imediatez, com isso surge a necessidade de uma ferramenta com o toda a informação relevante sobre o projeto, em uma interface gráfica amigável para o usuário.
